# 폰트 폴더

이 폴더에 한글 폰트 파일(.ttf)을 넣어서 사용할 수 있습니다.

## 권장 폰트

### 나눔고딕 (NanumGothic)
- `NanumGothic-Bold.ttf` (권장) - 더 굵고 선명함
- `NanumGothic.ttf`

### 설치 방법
1. 나눔고딕 다운로드: https://hangeul.naver.com/2017/nanum
2. 원하는 폰트 파일을 이 폴더에 복사
3. 파일명을 위의 권장 이름으로 변경 (선택사항)

## 폰트 파일 이름 우선순위

게임은 다음 순서로 폰트 파일을 찾습니다:
1. `NanumGothic-Bold.ttf` (가장 권장)
2. `NanumGothic.ttf`
3. `NanumBarunGothic-Bold.ttf`
4. `NotoSansKR-Bold.ttf`
5. `MalgunGothic.ttf` (Windows)

## 크로스 플랫폼 지원

- **Windows**: 맑은 고딕 자동 사용 (폰트 파일이 없어도 작동)
- **macOS**: AppleGothic 자동 사용 (폰트 파일이 없어도 작동)
- **Linux**: Noto Sans CJK KR 자동 사용

폰트 파일을 추가하면 모든 플랫폼에서 동일한 폰트를 사용할 수 있습니다.

## 주의사항

- 폰트 파일은 `.ttf` 형식만 지원합니다
- 폰트 파일이 없어도 시스템 폰트를 자동으로 사용합니다
- 상업적 사용 시 폰트 라이선스를 확인하세요
